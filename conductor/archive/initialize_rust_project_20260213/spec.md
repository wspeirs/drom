# Specification - Initialize Rust project and implement basic task execution

## Overview
This track involves initializing the Rust environment and implementing a minimal proof-of-concept for `drom` that can execute a single command defined in a configuration file.

## Requirements
- Initialize a new Rust project using `cargo init`.
- Define a simple TOML configuration format for tasks.
- Implement logic to parse the configuration and execute the specified command.
- Ensure basic error handling for missing files or execution failures.
